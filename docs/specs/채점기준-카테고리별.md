## 공통 폰트 규칙

- **국문**: 애터미 M (Medium) / 애터미 L (Light) / 애터미 B (Bold)
- **영문**: 애터미 M 또는 DINOT-M / DINOT-L
- **법규 표시사항(성분 등)**: Noto Sans CJK 허용
- **수출용 현지어**: 해당 언어에 맞는 폰트 허용 (단, 폰트 일관성 유지)
- **위반**: 세리프체, 손글씨체, Arial/굴림 등 시스템 기본 폰트, 혼합 폰트 패밀리

## 용기 종류별 기본 규칙

### 단박스 (Box)
- 컬러박스: 세로 전체의 1/3 또는 1/4 (카테고리별 상이)
- Line Box (뷰티·헤어바디 앱솔루트): 단박스 세로 길이의 33%, stroke 2.0~2.4pt
- 심볼 스케일: 가로폭 기준 20~35%
- 사방간격(상하좌우 여백) 통일 필수

### 용기 (병·자·튜브·펌프)
- **컬러박스 없음** — 가이드 정상 (감점 금지)
- BI 로고: 가로폭 기준 50% 이상
- 심볼 스케일: 가로폭 기준 25~50%
- 구분선: 제품명 세로 높이 기준 50% 위치, stroke 0.3~0.5pt

### 파우치
- 컬러박스: 하단, 세로의 1/4
- 심볼 스케일: 가로폭 기준 18~20%
- 사방간격 통일 필수

## 카테고리별 채점 기준

---

### 🔵 뷰티-앱솔루트 (Beauty Absolute)

> 전면: ATOMY ABSOLUTE(금박) + 라인명 + 영문품목명 + 카피 + 내용량  
> 후면: atomy美 심볼(블루) + 국/영문제품명 + 표시사항

#### ① 로고 및 심볼 (15점)
- BI 로고: **ATOMY ABSOLUTE** 금박 텍스트 필수 — "ABSOLUTE" 없으면 **0점**
- 금박은 사진에서 흰색·크림·밝은회색으로 보여도 금박으로 인정 (감점 금지)
- 전면에 atomy美 원형 심볼이 보이면 **0점** (심볼은 후면 전용)
- BI 로고 스케일: 가로폭 기준 50% 이상
- atomy美 심볼: 가로폭 기준 20~30% (후면)

#### ② 컬러박스 / Line Box (25점)
- **단박스**: Line Box 필수 — 앱솔루트 금박, stroke 2.0pt 이상, 단박스 세로의 33%
- **용기**: 컬러박스 없음 정상 — 구분선(0.3pt, 제품명 길이의 30%) 평가
- Line Box 내 라인명·품목명·영문카피 삽입 여부

#### ③ 제품명 타이포그래피 (25점)
- **크기 규칙**: 영문(8.5pt↑) ≥ 국문(5pt↑) — **영문이 크거나 같아야 함**
- 국문이 영문보다 크면 **가이드 위반**
- stroke: 0.2pt 이상
- 폰트: 애터미 M

#### ④ 필수 요소 (15점)
- ATOMY ABSOLUTE BI 로고 (금박)
- 라인명 (애터미 L, 7pt 이상)
- 영문 품목명 (LOTION·TONER 등)
- 카피 (애터미 L, 4.5pt 이상)
- 내용량 (애터미 L, 7pt 이상)
- 분리배출 마크 (가로 8mm 이상)

#### ⑤ 레이아웃 (20점)
- **정렬**: 로고·제품명·카피 모두 **가운데 정렬** 정상
- 사방간격 동일
- 구분선: 제품명 세로 높이 기준 50% 위치

### 🔵 뷰티-스탠다드 (Beauty Standard)

> 전면: ATOMY 로고 + 영문품목명 + 구분선 + 카피 + 내용량  
> 후면: atomy美 심볼

#### ① 로고 및 심볼 (15점)
- BI 로고: **ATOMY 텍스트** (블루 또는 화이트)
  - 흰색·밝은 용기 → **애터미 블루** (#00B5EF) 로고
  - 컬러·어두운 용기 → **흰색** 로고
  - 블루·화이트 외 색상 → 감점 + "본사 디자인팀 확인 필요" 표기
- **"ABSOLUTE" 텍스트가 보이면 → 0점 + 전체 불합격**
- 전면에 ATOMY 텍스트 로고 OR atomy美 심볼 둘 다 허용
- BI 로고 스케일: 가로폭 기준 50% 이상

#### ② 컬러박스 / 구분선 (25점)
- **단박스 (Bottle 타입)**: Line Box — 컨셉 컬러, stroke 1.7pt 이하
- **단박스 (Jar 타입)**: 화이트 Line Box, 가로폭 기준 75% 축소
- **용기**: 컬러박스 없음 정상 — 구분선(0.3pt, 제품명 30%) 평가

#### ③ 제품명 타이포그래피 (25점)
- **크기 규칙**: 영문(6pt↑) ≥ 국문(5pt↑) — **비슷하거나 영문이 약간 큼**
- 색상: 컨셉 컬러
- 라인명 폰트: 애터미 L, 6pt 이상, 애터미 다크그레이
- 카피 폰트: DINOT-L, 4.5pt 이상

#### ④ 필수 요소 (15점)
- ATOMY BI 로고
- 영문 품목명
- 구분선 (가로 0.3pt, 품목명 길이의 30%)
- 카피
- 내용량 (애터미 L, 6pt 이상)
- 분리배출 마크 (8mm 이상)

#### ⑤ 레이아웃 (20점)
- **정렬**: **가운데 정렬** 정상
- 사방간격 동일

### 🔵 뷰티-색조 (Beauty Color)

#### ④ 필수 요소 (15점)
